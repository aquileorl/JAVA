# JAVA - Ejercicios y Prácticas

Este repositorio contiene **ejercicios y proyectos de programación en Java** realizados durante las clases del curso de programación del año pasado.  
Incluye desde conceptos básicos hasta programación orientada a objetos, estructuras de datos y manejo de excepciones.

## Contenido principal
- **Variables y tipos de datos:** `1.string_args`, `2.pelicula`, `3.ejercicio_condicionales_1`  
- **Bucles y arrays:** `4.bucles`, `5.array`  
- **POO:** `7.poo`, `clasesAnidadas`, `herencias`  
- **Colecciones:** `ArrayList`, `ArrayList_`  
- **Excepciones:** `Excepciones`  
- **Proyectos prácticos:** `Proyecto1`, `Proyecto2/matematicas`, `Proyecto3/semana1`

## Objetivo
Servir como **repositorio de práctica y repaso** del temario de Java, útil para estudiantes y para demostrar conocimientos básicos y avanzados del lenguaje.

## Requisitos
- Java JDK 17 o superior
- IDE compatible (IntelliJ, VS Code, Eclipse)
